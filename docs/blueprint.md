# **App Name**: EnviroClean Training

## Core Features:

- Interactive Simulations: Interactive modules that visually demonstrate proper cleaning techniques for different hospital environments.
- Knowledge Checks: Quizzes and assessments to test understanding and retention of training materials.
- Personalized Feedback: AI-powered tool to analyze user performance in simulations and quizzes, providing personalized feedback and recommendations for improvement.

## Style Guidelines:

- Primary color: White to convey cleanliness and sterility.
- Secondary color: Shades of blue (baby blue, royal blue, sky blue) for a calming and trustworthy feel.
- Accent: Medium gray for neutral elements and backgrounds.
- Clear and readable fonts for easy comprehension.
- Use simple and intuitive icons to represent different cleaning tasks and environments.
- Clean and organized layout for easy navigation and information access.
- Subtle animations to guide users through interactive modules.
